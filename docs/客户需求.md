# 客户需求

代币总量为16亿枚，其中20%（3.2亿枚）用于IEO

阶段	代币数量（枚）	价格（美元）	募集资金（美元）
第一阶段	100,000,000	0.125	12,500,000
第二阶段	110,000,000	0.150	16,500,000
第三阶段	110,000,000	0.175	19,250,000
总计	320,000,000		51,250,000

价格差异分析：
第一阶段到第二阶段涨幅：（0.150 - 0.125）/ 0.125 = 20%
第二阶段到第三阶段涨幅：（0.175 - 0.150）/ 0.150 ≈ 16.7%


IEO结束后代币释放细节
IEO结束后，投资者获得的代币将采用“一次性释放 + 线性释放”的方式。具体方案如下：
一次性释放：在IEO结束后的7天内，立即释放投资者购买代币总量的 10%。
线性释放：剩余 90% 的代币将在IEO结束后的 12个月 内线性释放。线性释放从IEO结束后第8天开始，每月释放一次，每次释放90%÷12=7.5%。

推荐奖励方案（只奖励一级）
推荐奖励旨在激励现有投资者推荐新投资者参与IEO。方案仅奖励一级推荐（即直接推荐）。
奖励比例：推荐人可获得被推荐人购买金额的 5% 作为代币奖励。奖励代币的价值按被推荐人购买阶段的IEO价格计算。
奖励来源：奖励代币来自项目方的团队激励（占代币总量的其他部分），不占用本次IEO的3.2亿枚代币，以确保募集资金目标完整。
奖励释放：奖励代币在IEO结束后，与投资者代币的一次性释放部分一同发放，立即全额给予推荐人。
示例：投资者A推荐了投资者B。
投资者B在第二阶段成功购买了价值1,500美元的代币（按0.150美元/枚，购买了10,000枚代币）。
投资者A将获得的奖励为：$1,500 * 5% = $75。
按第二阶段的IEO价格0.150美元/枚计算，投资者A将获得 $75 / 0.150 = 500枚代币作为推荐奖励。
这500枚奖励代币将在IEO结束后7天内一次性发放给投资者A。
